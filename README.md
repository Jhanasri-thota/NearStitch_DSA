# NearStitch – Console Based Tailoring Service Booking System

## Project Overview
NearStitch is a console-based tailoring service booking application developed using **Java and Data Structures**.  
The system allows users to search nearby tailor shops, view shop details, select dress models, and book stitching services easily through a console interface.

This project demonstrates the practical implementation of **Data Structures concepts** such as LinkedList, Queue, and HashMap in a real-world scenario.

---

## Features
- Location and city selection
- Display tailor shop details (rating, address, contact)
- Tailor shop search functionality
- Dress model selection with stitching cost
- Customer booking system
- Pickup date and time scheduling
- Automatic Order ID generation
- Final booking receipt with cost and delivery details

---

## Data Structures Used

| Data Structure | Purpose |
|----------------|--------|
| LinkedList | Stores tailor shop details |
| Queue | Manages booking requests (FIFO order) |
| HashMap | Stores customer records using phone number |

---

## Example Workflow

1. User selects location and city  
2. System displays available tailor shops  
3. User searches or selects a tailor shop  
4. User chooses dress model for stitching  
5. Customer enters personal details  
6. System generates booking with order ID  
7. Final receipt with pickup and delivery details is displayed
